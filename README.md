# Pokédex DIO

Este projeto é um guia para criar uma Pokédex usando HTML, CSS e JavaScript, consumindo a PokéAPI.

## Objetivo do projeto

- Praticar Git/GitHub
- Praticar consumo de API REST
- Desenvolver habilidades de front-end (HTML, CSS e JS)

## Estrutura do projeto

1. **index.html**
   - Estrutura da página
   - Cabeçalho com input e botão
   - Área principal para mostrar Pokémon
   - Rodapé

2. **style.css**
   - Estilização da página e dos cards de Pokémon
   - Layout responsivo
   - Cores por tipo de Pokémon

3. **script.js**
   - Buscar Pokémon da API
   - Exibir informações na página
   - Event listeners para botão e Enter
   - Tratamento de erros e unidades

4. **Como testar**
   - Abrir `index.html` no navegador
   - Ou publicar usando GitHub Pages para ver online
